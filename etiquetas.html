¿Qué son las Etiquetas Semánticas?

Las etiquetas semánticas HTML5 proporcionan significado a las diferentes partes de una página web. A diferencia de las etiquetas no semánticas (como <div> y <span>) que no tienen significado por sí mismas, las etiquetas semánticas describen el propósito de su contenido, lo que mejora la legibilidad para desarrolladores y motores de búsqueda, y la accesibilidad para lectores de pantalla.

1. <article>

La etiqueta <article> se utiliza para envolver contenido independiente y autosuficiente. Piensa en ella como una entrada de blog, un artículo de noticias, un comentario de usuario, o cualquier pieza de contenido que podría distribuirse y tener sentido por sí misma fuera del contexto de la página principal.

Ejemplo de Uso:


                <article>
    <h2>Mi Primera Entrada de Blog</h2>
    <p>Este es el contenido de mi entrada, que puede leerse de forma independiente.</p>
    <footer>
        <p>Publicado el <time datetime="2025-05-22">22 de Mayo de 2025</time></p>
    </footer>
</article>

2. <section>

La etiqueta <section> agrupa contenido relacionado temáticamente. Sirve para dividir el contenido de una página en secciones lógicas, como una sección "Sobre Nosotros", "Productos", o "Contactos". Cada <section> debe tener idealmente un encabezado (<h1> a <h6>).

Ejemplo de Uso:


<section>
    <h2>Nuestros Servicios</h2>
    <p>Ofrecemos una variedad de servicios de desarrollo web.</p>
    <!-- Más contenido relacionado con servicios -->
</section>

3. <nav>

La etiqueta <nav> se utiliza para agrupar enlaces de navegación principal. Contiene los enlaces que son esenciales para navegar por el sitio web, como el menú principal, los enlaces a secciones importantes, o el índice de un documento largo.

Ejemplo de Uso:


<nav>
    <ul>
        <li><a href="/">Inicio</a></li>
        <li><a href="/productos">Productos</a></li>
        <li><a href="/contacto">Contacto</a></li>
    </ul>
</nav>

4. <header>

La etiqueta <header> representa contenido introductorio o un grupo de ayudas a la navegación. Puede contener encabezados (<h1> a <h6>), un logotipo, o incluso la etiqueta <nav> para el menú principal del sitio o de una sección.

Ejemplo de Uso:


<header>
    <h1>Nombre de Mi Sitio</h1>
    <nav>
        <!-- Enlaces de navegación -->
    </nav>
</header>

5. <footer>

La etiqueta <footer> representa el pie de página de su sección más cercana o de todo el documento. Generalmente contiene información como derechos de autor, enlaces a documentos relacionados, información de contacto, o enlaces a redes sociales.

Ejemplo de Uso:


<footer>
    <p>&copy; 2025 Mi Empresa. Todos los derechos reservados.</p>
    <a href="/politica-privacidad">Política de Privacidad</a>
</footer>
